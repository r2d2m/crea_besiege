## Very simple Besiege-like prototype using Unity.
![](https://github.com/vincent-breysse/CreaBesiege/blob/master/Screens/0.png)

INPUTS 
---------------------------------------------------------------------------------------------------------------------------

### EDITION
- Save the vehicle : S
- Open a vehicle : O
- Mouse Left to place block in your hand
- Mouse right to rotate around your vehicle

### IN GAME
- Rotate wheels : Z/S
- Use boosters : Space
- Mouse right to rotate around your vehicle
